# SimpleGA

Simple Genetic Algorithm base framework, supporting population, evaluation, mutators, crossover and next generation.

## Installation

Via npm on Node:

```
npm install simplega
```


## Usage

Reference in your program:

```js
var simplega = require('simplega');
```
